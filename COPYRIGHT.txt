2. Data

No third-party data is being distributed directly through the GitHub repository,
but the processing scripts will draw data from external resources. These files
are NOT subject to the GPLv3.

2.1 Geospatial data
Geospatial polygon data for country outlines is being drawn from the
Natural Earth project. This data is in the public domain, see
https://www.naturalearthdata.com/about/terms-of-use/ for details

2.2 Disease surveillance systems data
